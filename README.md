Welcome to my adapted version of the Klipper project!

To see the original, please visit:
https://www.klipper3d.org/

The Klipper firmware controls 3d-Printers. It combines the power of a
general purpose computer with one or more micro-controllers. See the
[features document](https://www.klipper3d.org/Features.html) for more
information on why you should use the Klipper software.


This version of Klipper software is Free Software as the original. See the [license](COPYING) or read
the [documentation](https://www.klipper3d.org/Overview.html).
